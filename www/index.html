Grain size analysis Android application


During my PhD I undertook a three month placement with petiole ltd, working from home, to develop an Android application that would allow plant science researchers to analyse a sample of grain or seed using their smartphone.


The application was prototyped in C++ before being made into an Android application written in Java. It uses the OpenCV computer vision library to calibrate the phone's camera, before counting and measuring a sample of grain or plant seed. The results can then be viewed on screen, or exported as a CSV file for further analysis.
